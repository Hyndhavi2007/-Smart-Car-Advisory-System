# -Smart-Car-Advisory-System
CarAdvisor 🚗✨

----------Purpose------------

CarAdvisor is a C++ program that helps people make smarter choices when they're shopping for a car.

Buying a car means weighing a lot of things—price, your budget, and the total cost over time. CarAdvisor steps in to handle some of that math for you and gives you straightforward advice to make the whole process easier.

----------Goal------------

The goal of this project is to assist a car buyer in understanding:

The estimated cost of a car

Budget considerations

Basic calculations involved in purchasing a vehicle.

---------Features------------

Takes user input (budget, preferences, etc.)

Performs basic calculations

Helps users evaluate car affordability

Simple and easy-to-use console interface

--------Technologies Used----------

C++
